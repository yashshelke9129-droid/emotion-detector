# Emotion Detector

This project is developed as part of the IBM Skills Network / Coursera Final Project.

## Features

* Detects emotions from text
* Returns:

  * Anger
  * Disgust
  * Fear
  * Joy
  * Sadness
* Identifies the dominant emotion
* Flask web interface
* Unit testing support

## Technologies Used

* Python
* Flask
* Requests
* Watson NLP
